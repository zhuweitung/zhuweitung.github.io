中级软考知识点三

中级软考知识点三

计算机网络

  • OSI七层模型
    • 应用层
    • 表示层
    • 会话层
    • 传输层：TCP，UDP
    • 网络层：路由器，三层交换机
    • 数据链路层：网桥，交换机，网卡
    • 物理层：中继器，集线器（Hub）
  • TCP/IP模型
    • 应用层：对应应用层，表示层和会话层
    • 传输层：对应传输层
    • 网际层：对应网络层
    • 网络接口层：对应数据链路层和物理层
  • UDP
    • SNMP
    • DNS
    • DHCP
    • TFTP
  • TCP
    • Telnet
    • SMTP
    • FTP
    • HTTP
    • POP3
  • IP地址
    • A类0~127.0~255.0~255.0~255，还有24位主机号
    • B类128~291.0~255.0~255.0~255，还有16位主机号
    • C类192~223.0~255.0~255.0~255，还有8位主机号
  • 数字签名
    • 发送方使用私钥加密生成签名
    • 接收方使用发送方的公钥解密验证签名
